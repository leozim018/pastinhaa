EXPLICAÇAO



  • Definindo a imagem de fundo:
    css
    Copy code
    body {
    background-image: url(https://i.pinimg.com/736x/99/a5/13/99a513baf777e9fe554465a207e738c8.jpg);
    }
    Essa regra define uma imagem de fundo para o corpo do documento. A imagem é carregada a partir do URL especificado.


  • Definindo margens e preenchimento para alguns elementos:
    css
    Copy code
    body, h1, p {
    margin: 0;
    padding: 0;
    }
    Essas regras definem margens e preenchimento zero para o corpo (body), títulos de nível 1 (h1) e parágrafos (p) no documento. Isso ajuda a remover espaçamento indesejado padrão aplicado pelo navegador.


  • Definindo a fonte padrão:
    css
    Copy code
    body {
    font-family: Arial, sans-serif;
    }
    Essa regra define a fonte padrão para o corpo do documento como "Arial" e, se não estiver disponível, qualquer fonte sans-serif.


  • Estilizando o cabeçalho:
    css
    Copy code
    header {
    background-color: #3ba726;
    color: #FFF;
    padding: 25px;
    text-align: center;
    }
    Essa regra define estilos para o elemento header. Ele terá um fundo verde (#3ba726), texto branco (#FFF), um preenchimento interno de 25 pixels e o texto centralizado.


  • Estilizando o conteúdo principal:
    css
    Copy code
    main {
    text-align: center;
    }
    Essa regra centraliza o texto no elemento main, que geralmente contém o conteúdo principal da página.


  • Estilizando o logo com animação:
    css
    Copy code
    .logo {
    max-width: 100%;
    animation: shake 5s ease-in 1;
    position: absolute;
    top: 25%;
    left: 40%;
    }
    Essa regra aplica estilos ao elemento com a classe .logo. Ele terá uma largura máxima de 100%, uma animação chamada shake que dura 5 segundos e inicia suavemente, e é posicionado absolutamente a 25% do topo e 40% da esquerda da janela.


  • Animação "shake" usando keyframes:
    css
    Copy code
    @keyframes shake {
    0% {
    transform: rotate(0);
    top: 0;
    left: 100%;
    }
    100% {
    top: 25%;
    left: 40%;
    transform: rotate(360deg);
    }
    }
    Essa regra define uma animação chamada shake usando keyframes. Ela faz o elemento animado girar 360 graus e se mover de uma posição inicial até 25% do topo e 40% da esquerda.


  • Estilizando o rodapé com animação:
    css
    Copy code
    footer {
    text-align: center;
    padding: 10px;
    color: rgb(5, 4, 4);
    animation: shake1 5s ease-in 1;
    position: absolute;
    top: 85%;
    left: 40%;
    }
    Essa regra aplica estilos ao elemento footer. Ele terá texto centralizado, um preenchimento interno de 10 pixels, cor de texto escura e uma animação chamada shake1 que dura 5 segundos e inicia suavemente. O rodapé é posicionado a 85% do topo e 40% da esquerda da janela.

    op: 0;
    left: 0%;
